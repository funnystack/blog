三十而立

圣诞节已过，2017也即将进入尾声

这一年学到了不少新技术

  • VUE 2.0
  • Spring Boot
  • Spring Cloud

这一年看了几本书

  • 淘宝这十年
  • 京东技术解密
  • 多线程与设计模式
  • App开发运维和架构实践
  • 许我向你看

这一年追了几部剧

  • 权利的游戏1-7(追剧真心累)
  • 亮剑(亮剑精神值得学习)
  • 猎场(职场就是猎场)
  • 人民的民义
  • 水浒传
  • 三生三世十里桃花
    不能再看剧了,实在是耽误时间。

这一年改变很多

  • 新发型，学会打理自己
  • 6-9月份的持续三个月的腰椎疼痛，使我意识到必须坚持锻炼，身体才是革命的本钱
  • 报考人民大学在职研究生为35之后的自己投资

这一年的家庭作业

  • 重游天津之眼
  • 越南芽庄之旅

这一年失去了很多

  • 媳妇的姥姥在腰疼2个月之后检查出是原发性骨髓瘤，没能走过这个冬天。她是我生命中的贵人，当初在在她父母不同意的情况下，她第一个同意我和媳妇的这段婚姻。
  • 大学室友景湘因为年轻的时候的车祸留下的脑水肿发作，在ICU艰难抗争了五个月之后，还是去了另外一个世界。

人生无常，生命是如此脆弱，那就不如享受当下吧。以前总是对未来充满各种期望，现在我虽然还是充满期望，但是那已经不再是全部。
2018即将拉开序幕，我也即将迈入了而立之年。
更坦然面对生活、更坦然面对工作。
